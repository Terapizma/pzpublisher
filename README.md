## 📖 Our Story & Vision
For years, the Project Zomboid community had to struggle with manual file copying and Steam's complex tools to upload or update mods. **PZPUBLISHER** was designed specifically to put an end to this struggle. Our goal was to provide mod developers with the smooth, modern, and sleek experience they deserve. After nights of coding and SteamWorks integration testing, we built this framework where you can manage complex Workshop tasks from a single interface. This tool is a professional signature left to the community by TN Studio.

## 🚀 Key Features

* 📦 **One-Click Workshop Publishing:** Select your mod folder, enter the details, and let PZPUBLISHER handle the rest. Steam uploads are handled instantly in the background.
* 🖼️ **Advanced Cover Editor:** Update Workshop images instantly through the app, keeping your mod's storefront always fresh.
* 📥 **SteamCMD Integration:** Uses Valve's official tools (SteamCMD) in the background to ensure the most secure and stable data transfer.
* 🎮 **SteamWorks Integration:** The app syncs with your Steam profile upon launch and provides direct access to your workshop items (likes, subscriber count, file size).
* 🎨 **Modern UI:** A professional and user-friendly design that matches the Project Zomboid atmosphere without straining the eyes.

## 💻 Technologies Used
This project is built on modern and stable infrastructures:
* **[Electron.js](https://electronjs.org/):** Powerful desktop application framework.
* **[Node.js](https://nodejs.org/):** Background operations and asynchronous file management.
* **[SteamWorks.js](https://github.com/ceifa/steamworks.js):** Steam API and Workshop integration.
* **Valve Core Tools:** SteamCMD integrations.
